What does Visualize do?

This app will be used by the University of Alabama Baseball program to aid in tours for Freshman and Sophomore baseball recruits. The app provides videos and information on the different facilities within Sewell-Thomas Stadium, such as the locker rooms, team lounge, batting cages, etc. This app will also include a map feature to help guide the user through Sewell-Thomas Stadium.

What features does it include?

This app has three main features: scan, map, and tour. The scan feature allows the users to scan QR codes that link to youtube videos about different areas withing Sewell-Thomas Stadium. The map feature opens a map of the stadium that the user can interact with. The tour feature goes through the different locations of the tour and provides information about each without the user having to scan QR codes.
